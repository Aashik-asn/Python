SNAKE – HOW TO RUN

Install Python 3.9 or above
https://www.python.org/downloads/

Install required library:
pip install pygame

Install the font:
Extract the "born2bsporty-fs.zip" and install Born2bSportyFS.ttf

Run the game:
python main.py

CONTROLS:
W A S D → Move
P → Pause
R → Restart
T → Change Theme
Enjoy :)
